# Strange Signals — Local/Private Notes

## Dev Environment
- Serve locally: `python3 -m http.server 8000` or `python3 proxy.py` (port 8080, includes API proxy)
- No build step — edit files directly, refresh browser
- Data files live in `data/` (gitignored, generated by pipeline)

## Data Pipeline
- Run `bash setup_sightings.sh` to download raw datasets and generate JSON
- Requires: Python 3, pandas, openpyxl
- Output: `data/sightings_map_data.json` (~15-20MB)

## Branch Strategy
- `main` — stable releases
- `claude/*` — Claude Code working branches
